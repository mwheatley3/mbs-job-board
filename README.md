

## Getting Started

```
$ cd my-project
$ npm install
$ bower install
$ npm start
```

Now visit [localhost:4000](http://localhost:4000/)

### Running the Tests

There is a basic test framework in your `test/` folder. To run the tests, simply run `npm test`.
